# Client Brief: Dodo Nutrition

> **Document Type**: Discovery Notes  
> **Status**: Reconstructed from project context  
> **Date**: 2026-01

---

## Business Overview

**Client**: Dodo Nutrition  
**Industry**: E-commerce / Supplements & Sports Nutrition  
**Market**: Tunisia (local only)

---

## Business Goals

1. **Primary**: Create a modern e-commerce platform for sports nutrition products
2. **Secondary**: Build brand recognition in the Tunisian fitness market
3. **Long-term**: Establish as the go-to online store for supplements in Tunisia

---

## Target Audience

| Segment       | Profile                                |
| ------------- | -------------------------------------- |
| **Primary**   | Fitness enthusiasts, gym-goers (18-35) |
| **Secondary** | Athletes, bodybuilders                 |
| **Tertiary**  | Health-conscious consumers             |

**Location**: Greater Tunis (Tunis, Ben Arous, Manouba, Ariana) + National

---

## Key Requirements (Non-negotiables)

| Requirement         | Reason                            |
| ------------------- | --------------------------------- |
| **TND Currency**    | Local market compliance           |
| **COD Payment**     | Primary payment method in Tunisia |
| **Dark Theme**      | Brand identity, premium feel      |
| **French + Darija** | Tunisian user experience          |
| **Mobile-First**    | Primary access method             |

---

## Competitors

- Physical supplement stores (local)
- Facebook/Instagram sellers
- International sites (shipping issues)

**Differentiation**: Professional e-commerce experience, local trust, COD support.
