# User Stories

> **Project**: Dodo Nutrition  
> **Format**: As a [Role], I want to [Action], so that [Benefit]

---

## Customer Stories

### Browsing

- **US-001**: As a customer, I want to browse products by category, so that I can find supplements for my goals.
- **US-002**: As a customer, I want to search for products, so that I can quickly find what I need.
- **US-003**: As a customer, I want to filter products by price, brand, and certification, so that I can narrow down options.
- **US-004**: As a customer, I want to see featured products, so that I can discover popular items.

### Product Details

- **US-010**: As a customer, I want to see product details including description, price, and nutritional info, so that I can make an informed decision.
- **US-011**: As a customer, I want to see product reviews, so that I can trust the product quality.
- **US-012**: As a customer, I want to see related products, so that I can discover more options.

### Cart & Checkout

- **US-020**: As a customer, I want to add products to my cart, so that I can purchase multiple items.
- **US-021**: As a customer, I want to adjust quantities in my cart, so that I can change my order.
- **US-022**: As a customer, I want to checkout with COD, so that I can pay on delivery.
- **US-023**: As a guest, I want to checkout without creating an account, so that I can purchase quickly.

### Account

- **US-030**: As a customer, I want to sign up/login, so that I can track my orders.
- **US-031**: As a customer, I want to view my order history, so that I can see past purchases.
- **US-032**: As a customer, I want to save products to a wishlist, so that I can buy them later.

### Reviews

- **US-040**: As a customer, I want to submit a review, so that I can share my experience.
- **US-041**: As a customer, I want to see overall ratings, so that I can quickly assess products.

### AI Assistant

- **US-050**: As a customer, I want to chat with an AI assistant, so that I can get product recommendations.
- **US-051**: As a customer, I want the AI to search products for me, so that I can find items easily.

## Admin Stories

### Dashboard

- **US-100**: As an admin, I want to see revenue and order stats, so that I can monitor business health.
- **US-101**: As an admin, I want AI-generated insights, so that I can identify trends and issues.

### Orders

- **US-110**: As an admin, I want to view all orders, so that I can manage fulfillment.
- **US-111**: As an admin, I want to update order status, so that I can track order progress.
- **US-112**: As an admin, I want to filter orders by status, so that I can focus on pending orders.

### Inventory

- **US-120**: As an admin, I want to see product stock levels, so that I can manage inventory.
- **US-121**: As an admin, I want to update product details, so that I can keep information current.

### Reviews

- **US-130**: As an admin, I want to approve/reject reviews, so that I can moderate content.
- **US-131**: As an admin, I want to feature reviews, so that I can highlight positive feedback.
- **US-132**: As an admin, I want to assign reviews to categories, so that they appear on relevant products.

---

## System Stories

- **SYS-001**: As the system, I want to decrement stock on order, so that inventory stays accurate.
- **SYS-002**: As the system, I want to send order confirmation emails, so that customers are notified.
- **SYS-003**: As the system, I want to validate cart before checkout, so that stale data is prevented.
